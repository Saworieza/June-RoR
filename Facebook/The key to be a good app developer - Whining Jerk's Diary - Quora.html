The key to be a good app developer

Vysakh Sreenivasan
2 upvotes by
Yuvaraja Balamurugan and
Pragadheesh Chander.
  • the number of different types of apps you develop.
  • you automate everything.

What happens if you don't save the time right from the navigation you make in your text editor to the test that checks if your code works, you need to automate? Time waste.
 What happens if you waste time? You don't develop number of different types of apps.
What happens if you don't enough apps? You now lost the key to be a good app developer.

You need to try to develop more apps. Why? You want to get the key to be a good developer.
You need to save more time in your develop flow. Why? You need to try to develop more apps.
You need to automate your development flow - testing, test editing, every other things. Why? You need to save more time in your develop flow.
Now you got the key to be a good developer.
